ABOUT

I wrote my first computer program and built my first website when I was 13 years old. In high school, my graphing calculator had so many custom programs and functions that teachers wouldn't let me use it on tests. Throughout college I programmed several projects, including a four-wheeled robot that navigates a maze using bumpers and a light sensor to reach a light source in the center. I worked for a while as an engineer, but found that it was not challenging me as it should. Each project was merely a recycled version of something done a thousand times before, in a field where very little has fundamentally changed for nearly a century. Tired of going through the same motions on each new project, I left my career in electrical engineering to pursue my true passion of software development.
